Select 步骤：人 + AI 协作，零脚本

用户在 Cursor 中查看候选话题表格，告诉 AI "选 1、3、5"，AI 直接写入 selected.json。全程对话完成，不需要跑任何 Python 命令。

输出规范

对齐 nexu-landing 博客仓库的 Astro 格式

每篇文章交付 4 个文件

{slug}.md ← 英文版（Astro frontmatter）
{slug}.zh.md ← 中文版（Astro frontmatter）
blog-{slug}-en.webp ← 英文封面
blog-{slug}-zh.webp ← 中文封面

Astro Frontmatter 格式

---
publishDate: 2026-03-31T00:00:00Z
title: "文章标题"
excerpt: "150-160 字符摘要"
image: ~/assets/images/blog-{slug}-{lang}.webp
tags:
  - announcements | guides | use-cases
category: Announcements | Guides | Use Cases
---
